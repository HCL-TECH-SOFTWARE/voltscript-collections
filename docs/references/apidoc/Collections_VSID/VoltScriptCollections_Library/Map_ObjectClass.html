      • unlock

        Public Sub unlock()
        Unlocks the Map, so put, remove, replace and reverse functions can be used again
         
        Code Sample
        Dim map as New Map("INTEGER", Nothing, False)
        Call map.put("Elem1", 1)
        Call map.put("Elem2", 2)
        Call map.put("Elem3", 3)
        Call map.lock()
        Try
            Call map.put("Elem4", 4)
        Catch
            If Err = 403 Then Print "Map was locked"
        End Try
        Call map.unlock()
        Call map.put("Elem4", 4)    ' Successful
